## Closeout Questions

Ask these before ending a serious run:

1. What did the user prefer about how this was planned, built, verified, or explained?
2. What did the assistant get wrong, overclaim, forget, or leave stale?
3. What reusable project pattern showed up?
4. What safeguard would prevent the biggest issue next time?
5. What should be promoted to the memory index or optional Vector Memory?
6. What should stay private, be rejected, or be deleted?

## Kickoff Questions

Ask these before planning the next serious run:

1. Which approved memories are relevant to this project?
2. Which memories are irrelevant or risky to apply here?
3. What should be done differently because of past runs?
4. Which safeguards should be checked before building?
